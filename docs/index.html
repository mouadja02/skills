Quick install 
curl -fsSL https://raw.githubusercontent.com/mouadja02/skills/main/install.sh \
  | bash -s -- <selector> -d <destination>
$content = irm https://raw.githubusercontent.com/mouadja02/skills/main/install.ps1
iex $content
Install-Skill <selector> -Dest <destination>

<selector> is a single skill (engineering-craft/test-driven-development), a category (engineering-craft), or a quoted glob ("*bmad*", "ai-agents/*"). Click any card below — the command pre-fills automatically.
