# marketing-and-growth

A complete **marketing operating system** — strategy, content, SEO, paid acquisition, and conversion-rate optimization. Most skills read shared context from a `marketing-context.md` document, so set that up first.

## Skills in this category

### Strategy & operations

| Skill | What it does |
| --- | --- |
| [`marketing-context`](./marketing-context/SKILL.md) | The foundational marketing context document all marketing skills read. ICP, positioning, brand voice, target audience, style guide. Run this first. |
| [`marketing-ops`](./marketing-ops/SKILL.md) | Central router for the marketing skill ecosystem. Use when unsure which marketing skill to invoke or when orchestrating multi-skill campaigns. |
| [`marketing-strategy-pmm`](./marketing-strategy-pmm/SKILL.md) | Product marketing — positioning, GTM strategy, competitive intelligence, product launches, ICP definition. |
| [`marketing-demand-acquisition`](./marketing-demand-acquisition/SKILL.md) | Demand-gen for Series A+ startups — paid ad spend optimization (LinkedIn, Google, Meta), SEO, partnership programs. |
| [`marketing-psychology`](./marketing-psychology/SKILL.md) | Apply psychological principles to marketing — cognitive bias, persuasion, behavioral science, decision-making frameworks. |
| [`launch-strategy`](./launch-strategy/SKILL.md) | Plan product launches and feature announcements — Product Hunt, beta launches, waitlists, GTM coordination. |

### Content & copy

| Skill | What it does |
| --- | --- |
| [`content-production`](./content-production/SKILL.md) | Full content pipeline from blank page to publish-ready piece. Triggers: "write a post about", "draft an article", "create content for". |
| [`content-humanizer`](./content-humanizer/SKILL.md) | Make AI-generated content sound genuinely human — kill clichés, add personality, restore voice. |
| [`cs-copywriting`](./cs-copywriting/SKILL.md) | Write/rewrite marketing copy for any page — homepage, landing, pricing, feature, about, product pages. |
| [`cs-cold-email`](./cs-cold-email/SKILL.md) | B2B cold outreach — prospecting emails, SDR sequences, sales emails to people who haven't asked to hear from you. |
| [`email-sequence`](./email-sequence/SKILL.md) | Drip campaigns, nurture sequences, onboarding emails, automated lifecycle email programs. |

### Channels (social & SEO)

| Skill | What it does |

### Paid & creative

| Skill | What it does |

### Conversion-rate optimization (CRO)

| Skill | What it does |
| --- | --- |
| [`page-cro`](./page-cro/SKILL.md) | General CRO for any marketing page — homepage, landing, pricing, feature, blog. "This page isn't converting" / "improve conversions". |
| [`signup-flow-cro`](./signup-flow-cro/SKILL.md) | Optimize signup, registration, account creation, free-trial activation. Reduce dropoff, friction, and form abandonment. |
| [`onboarding-cro`](./onboarding-cro/SKILL.md) | Optimize post-signup activation, first-run experience, time-to-value, empty states. |

> **A/B testing:** for hypothesis writing, sample-size calculation, and statistical interpretation, use [`product-management/experiment-designer`](../product-management/experiment-designer/) — it's the canonical home for experiment methodology and now covers both product *and* marketing/CRO contexts.

## Suggested workflow

```
marketing-context  →  marketing-strategy-pmm  →  launch-strategy  →
  content-production / paid-ads / cs-cold-email  →
    page-cro / signup-flow-cro  →  experiment-designer  →  campaign-analytics
```

When unsure which skill, invoke `marketing-ops` as the router.
