# microsoft-agents

Skills for **Microsoft Copilot agents, declarative agents, Foundry, and Azure AI agent tooling**.

## Skills in this category

| Skill | What it does |
| --- | --- |
| `microsoft-agent-framework` | Microsoft agent framework patterns — plugins, planners, orchestration. |
| `declarative-agents` | Build Microsoft 365 declarative agents with manifest and capabilities. |
| `foundry-agent-sync` | Sync agents with Azure AI Foundry. |
| `workiq-copilot` | WorkIQ Copilot integration patterns. |
| `entra-agent-user` | Entra ID identity for agents — user-on-behalf-of, managed identity. |
| `hosted-agents` | Deploy and manage hosted agents on Azure. |
| `from-the-other-side-vega` | Advanced Microsoft agent patterns from the Vega framework. |
| `mcp-copilot-studio-server-generator` | Generate MCP servers for Copilot Studio. |
| `mcp-create-adaptive-cards` | Create Adaptive Cards for Copilot agents via MCP. |
| `mcp-create-declarative-agent` | Create declarative agent manifests via MCP. |
| `mcp-deploy-manage-agents` | Deploy and manage agents via MCP tooling. |

## When to use this category

- "Build a Microsoft 365 Copilot agent"
- "Create a declarative agent"
- "Connect to Azure AI Foundry"
- "Generate MCP tools for Copilot Studio"
- "Add Entra ID auth to my agent"
