# react-frontend

Skills for **React, Vue, Next.js, and modern frontend frameworks** — including React 18/19 migration, mobile (Flutter, React Native), and framework-specific experts.

## Skills in this category

### React 18 migration

| Skill | What it does |
| --- | --- |
| `react18-batching-patterns` | Automatic batching in React 18 — upgrade patterns. |
| `react18-dep-compatibility` | Check and fix dependency compatibility for React 18. |
| `react18-enzyme-to-rtl` | Migrate Enzyme tests to React Testing Library. |
| `react18-legacy-context` | Migrate legacy Context API to modern context. |
| `react18-lifecycle-patterns` | Lifecycle method migration — componentDidMount → useEffect etc. |
| `react18-string-refs` | Remove string refs, migrate to createRef / useRef. |

### React 19 patterns
