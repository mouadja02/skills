# framework-flutter-expert

Comprehensive guidelines and best practices for developing mobile applications with Flutter, Dart, and the BLoC state management pattern.

## Install

With the [GitHub CLI](https://cli.github.com/) (v2.90.0+):

```bash
gh skill install mouadja02/skills engineering-craft/framework-flutter-expert
```

Works with Claude Code, Cursor, Codex, OpenCode, Gemini CLI, Windsurf, and [many more agents](https://cli.github.com/manual/gh_skill_install). Add `--scope user` to install across every project for your current agent, or `--agent claude-code` to target a specific agent.

For other install methods (Claude Code plugin marketplace, Cursor Rules, etc.) see the [root README](../../../README.md#install-skills-no-full-clone).

## What it covers

See [SKILL.md](SKILL.md) for the full reference, including:

- Clean architecture and dependency injection with GetIt
- Proper BLoC patterns and state management
- Error handling with the Either type
- GoRouter implementation and widget best practices
