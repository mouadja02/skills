# business-strategy

Skills for **C-suite advisory, executive decision support, company operating system, and strategic planning**.

## Skills in this category

### C-suite advisors

| Skill | What it does |
| --- | --- |
| `ceo-advisor` | CEO advisory — vision, strategy, culture, stakeholder management. |
| `cfo-advisor` | CFO advisory — financial planning, unit economics, fundraising. |
| `cto-advisor` | CTO advisory — tech strategy, team building, architecture decisions. |
| `cmo-advisor` | CMO advisory — brand, demand gen, positioning, marketing strategy. |
| `coo-advisor` | COO advisory — operations, processes, scaling, execution. |
| `cpo-advisor` | CPO advisory — product strategy, roadmap, discovery. |
| `cro-advisor` | CRO advisory — revenue operations, sales strategy, GTM execution. |
| `chro-advisor` | CHRO advisory — talent, culture, compensation, org design. |
| `ciso-advisor` | CISO advisory — security strategy, risk, compliance. |
| `c-level-advisor` | General C-level advisor — strategic decisions across functions. |
| `chief-of-staff` | Chief of Staff — agenda, priorities, decision tracking, comms. |
| `executive-mentor` | Executive mentoring — leadership, communication, presence. |

### Board & company

| Skill | What it does |
