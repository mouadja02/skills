# dotnet

Skills for the **.NET / C# ecosystem** — C# patterns, WinUI, MVVM, WinApp, NuGet, and VS Code extension development.

## Skills in this category

### C# language

| Skill | What it does |
| --- | --- |
| `csharp-async` | Async/await patterns, Task, cancellation, deadlock prevention. |
| `csharp-docs` | XML documentation comments, IntelliSense docs, doc generation. |
| `csharp-mstest` | Unit testing with MSTest — test structure, assertions, mocking. |
| `csharp-nunit` | Unit testing with NUnit — attributes, assertions, fixtures. |
| `csharp-tunit` | Unit testing with TUnit — modern parallelized testing. |
| `csharp-xunit` | Unit testing with xUnit — facts, theories, fixtures. |

### .NET framework & ecosystem

| Skill | What it does |
| --- | --- |
| `dotnet-best-practices` | .NET architecture, SOLID, DI, performance patterns. |
| `dotnet-design-pattern-review` | Review .NET code for design pattern correctness. |
| `dotnet-timezone` | Timezone handling with DateTimeOffset, TimeZoneInfo, NodaTime. |
| `dotnet-upgrade` | Migrate from .NET Framework to modern .NET — upgrade assistant guidance. |
| `nuget-manager` | NuGet package management, versioning, private feeds, lock files. |

### MVVM / Windows UI

| Skill | What it does |
| --- | --- |
| `mvvm-toolkit` | MVVM Toolkit — ObservableObject, RelayCommand, Messenger. |
| `mvvm-toolkit-di` | Dependency injection in MVVM Toolkit apps. |
| `mvvm-toolkit-messenger` | WeakReferenceMessenger, StrongReferenceMessenger patterns. |
| `winapp-cli` | Windows App SDK CLI tooling. |
| `winmd-api-search` | Search and explore WinMD API metadata. |
| `winui3-migration-guide` | Migrate UWP or WPF to WinUI 3. |

### VS Code extensions

| Skill | What it does |
| --- | --- |
| `vscode-ext-commands` | VS Code extension commands, activation events, keybindings. |
| `vscode-ext-localization` | Localize VS Code extensions — l10n, bundle files, multi-language. |

- "Write C# async code" / "unit test in xUnit / MSTest / NUnit"
