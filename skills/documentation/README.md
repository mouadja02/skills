# documentation

Skills for **writing, generating, and maintaining technical documentation** — READMEs, ADRs, API docs, project docs, and markdown tooling.

## Skills in this category

### README & project docs

| Skill | What it does |
| --- | --- |
| `create-readme` | Generate a comprehensive README from a codebase. |
| `readme-writer` | Write clear, engaging README files. |
| `readme-blueprint-generator` | Generate README blueprints for different project types. |
| `project-docs` | Create and maintain project documentation. |
| `project-development` | Document a project's development process and decisions. |

### Architecture & decision records

| Skill | What it does |
| --- | --- |
| `create-architectural-decision-record` | Create ADRs — context, decision, consequences. |
| `architecture-blueprint-generator` | Generate architecture blueprints from code or specs. |
| `code-exemplars-blueprint-generator` | Generate code exemplar documentation. |

### API & code documentation

| Skill | What it does |
| --- | --- |
| `documentation-writer` | Write technical documentation — guides, references, tutorials. |
| `oo-component-documentation` | Document object-oriented components — classes, interfaces, patterns. |
| `comment-code-generate-a-tutorial` | Transform commented code into a tutorial. |
| `microsoft-code-reference` | Microsoft-style code reference documentation. |
| `microsoft-docs` | Write documentation following Microsoft Docs standards. |

### LLMs.txt & AI-friendly docs

| Skill | What it does |
| --- | --- |
| `create-llms` | Create llms.txt files for AI agent consumption. |
| `update-llms` | Update an existing llms.txt with new content. |

### Markdown & conversion

| Skill | What it does |
| --- | --- |
| `convert-plaintext-to-md` | Convert plain text to well-structured Markdown. |
| `markdown-to-html` | Convert Markdown to HTML with styling. |
| `md-to-docx` | Convert Markdown to Word (.docx) documents. |
| `mkdocs-translations` | Add translations to MkDocs documentation sites. |
| `update-markdown-file-index` | Update index/TOC in Markdown file collections. |

### Blueprints & structure docs

| Skill | What it does |
| --- | --- |
| `folder-structure-blueprint-generator` | Document and generate project folder structures. |
| `project-workflow-analysis-blueprint-generator` | Analyze and document project workflows. |
| `technology-stack-blueprint-generator` | Generate technology stack documentation. |

### File & media conversion _(from steipete/agent-scripts)_
