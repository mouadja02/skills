# product-management

Skills for **product managers, product owners, and product leaders** — discovery, prioritization, experimentation, UX research, and agile delivery.

These complement [`business-and-strategy/cpo-advisor`](../business-and-strategy/cpo-advisor/) (strategic product leadership) with hands-on PM execution skills.

## Skills in this category

### Strategy & vision

| Skill | What it does |
| --- | --- |
| [`product-strategist`](./product-strategist/SKILL.md) | Strategic toolkit for Heads of Product — OKR cascade generation, quarterly planning, competitive landscape analysis, product vision documents, team scaling proposals. |
| [`product-manager-toolkit`](./product-manager-toolkit/SKILL.md) | Comprehensive PM toolkit — RICE prioritization, customer interview analysis, PRD templates, discovery frameworks, GTM strategies. |
| [`growth-pm-plg`](./growth-pm-plg/SKILL.md) | Product management frameworks focusing on Product-Led Growth (PLG), AARRR funnels, viral loops, and user retention mechanics. |

### Discovery & research

| Skill | What it does |
| --- | --- |
| [`product-discovery`](./product-discovery/SKILL.md) | Validate opportunities, map assumptions, plan discovery sprints, test problem-solution fit *before* committing delivery resources. |
| [`ux-researcher-designer`](./ux-researcher-designer/SKILL.md) | UX research and design — data-driven persona generation, journey mapping, usability testing frameworks, research synthesis. |
| [`experiment-designer`](./experiment-designer/SKILL.md) | Plan, design, and interpret any A/B test, split test, or multivariate experiment — for products *or* marketing pages. Hypothesis writing, sample-size calculator, ICE prioritization, traffic allocation, statistical-significance analysis, guardrail metrics. The canonical home for experiment methodology. |

### Analytics & measurement

| Skill | What it does |
| --- | --- |
| [`product-analytics`](./product-analytics/SKILL.md) | Define product KPIs, build metric dashboards, run cohort/retention analysis, interpret feature adoption trends. |

### Agile delivery

| Skill | What it does |
| --- | --- |
| [`agile-product-owner`](./agile-product-owner/SKILL.md) | Backlog management, user-story writing, acceptance criteria, sprint planning, velocity tracking. |
| [`scrum-master`](./scrum-master/SKILL.md) | Data-driven Scrum coaching — sprint planning, velocity tracking, retrospectives, standups, backlog grooming, blocker resolution. |
| [`roadmap-communicator`](./roadmap-communicator/SKILL.md) | Roadmap narratives, release notes, changelogs, stakeholder updates tailored for executives, engineering teams, and customers. |

## Suggested workflow

```
product-strategist (set vision)  →  product-discovery (validate)  →
  ux-researcher-designer (research)  →  product-manager-toolkit (prioritize)  →
    experiment-designer (test)  →  agile-product-owner / scrum-master (deliver)  →
      product-analytics (measure)  →  roadmap-communicator (communicate)
```
