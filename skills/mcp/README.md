# mcp

Skills for **building Model Context Protocol (MCP) servers** across all major languages, plus MCP tooling and security.

## Skills in this category

### MCP tooling

| Skill | What it does |
| --- | --- |
| `mcp-server-builder` | Build MCP servers from scratch — tool definitions, resource handling, protocol conformance. |
| `mcp-cli` | Use the MCP CLI to manage, test, and deploy MCP servers. |
| `mcp-security-audit` | Security audit for MCP servers — injection, access control, data leakage. |
