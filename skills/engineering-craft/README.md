# engineering-craft

Skills for **senior engineering roles, general coding practices, and cross-cutting technical concerns** that don't fit a specific language or framework.

## Skills in this category

### Senior IC roles

| Skill | What it does |
| --- | --- |
| `senior-architect` | Senior architect perspective — system design, trade-offs, ADRs. |
| `senior-backend` | Senior backend engineer — API design, scalability, data patterns. |
| `senior-frontend` | Senior frontend engineer — performance, accessibility, design systems. |
| `senior-fullstack` | Senior fullstack engineer — end-to-end feature development. |
| `senior-data-engineer` | Senior data engineer — pipelines, warehouses, quality, governance. |
| `senior-data-scientist` | Senior data scientist — modeling, experimentation, production ML. |
| `senior-ml-engineer` | Senior ML engineer — training, serving, MLOps, monitoring. |
| `senior-pm` | Senior PM perspective — product strategy, discovery, metrics. |
| `senior-qa` | Senior QA engineer — test strategy, automation, quality gates. |

### Coding practices & mindset

| Skill | What it does |
| --- | --- |
| `karpathy-guidlines` | Andrej Karpathy's coding guidelines — simplicity, clarity, taste. |
| `tech-debt-tracker` | Track and prioritize technical debt systematically. |
| `tech-stack-evaluator` | Evaluate and compare technology stacks. |
| `write-coding-standards-from-file` | Generate coding standards from existing codebase examples. |
| `ai-code-migrator` | Migrate codebases using AI — pattern recognition, bulk changes. |

### Planning & execution

| Skill | What it does |
| --- | --- |
| `writing-plans` | Write implementation and project plans. |
| `executing-plans` | Execute plans systematically — task decomposition, tracking. |
| `dispatching-parallel-agents` | Dispatch multiple agents in parallel for large tasks. |
| `subagent-driven-development` | Use subagents to drive development across large codebases. |
| `brainstorming` | Structured brainstorming for technical decisions. |
| `using-superpowers` | Activate and chain skills for complex multi-domain tasks. |

### Mentoring & onboarding

| Skill | What it does |
| --- | --- |
| `code-tour` | Create guided code tours for new contributors. |
| `codebase-onboarding` | Onboard new engineers to a codebase systematically. |
| `mentoring-juniors` | Mentor junior engineers — feedback, growth, code reviews. |
| `add-educational-comments` | Add educational inline comments to explain complex code sections. |

## When to use this category

- "I need a senior architect / backend / ML engineer perspective"
- "Track technical debt" / "evaluate tech stacks"
- "Plan and execute a large implementation"
- "Mentor a junior engineer" / "onboard to a codebase"
- "Salesforce development" / "game engine patterns"
