# prompting

Skills for **prompt engineering, optimization, and creative ideation** — write better prompts, improve AI outputs, and apply systematic thinking frameworks.

## Skills in this category

### Prompt engineering

| Skill | What it does |
| --- | --- |
| `senior-prompt-engineer` | Prompt optimization, A/B testing, versioning, governance, structured-output design, few-shot examples. |
| `prompt-master` | Generate optimized, production-ready prompts for any AI tool (LLMs, Cursor, Midjourney, coding agents). |
| `prompt-builder` | Interactive prompt builder — guided prompt construction for any use case. |
| `boost-prompt` | Improve an existing prompt — clarity, specificity, context, chain-of-thought. |
| `finalize-agent-prompt` | Polish and finalize agent system prompts before production. |
| `tldr-prompt` | Condense long prompts to their essential instruction. |
| `ai-prompt-engineering-safety-review` | Review prompts for safety, bias, security vulnerabilities, and effectiveness. |

### Creative thinking & ideation

| Skill | What it does |
| --- | --- |
| `bold-ideas-generator` | Generate bold, unconventional, high-impact ideas — bans incremental thinking. |
| `cognitive-frame-shift` | Analyze any problem through 6 radical lenses (child, alien, expert, contrarian, future self, hostile user). |
| `creative-constraints` | Use artificial limitations as creative fuel — SCAMPER, forced associations. |
| `deep-focus-protocol` | Structure work sessions for peak cognitive performance on hard problems. |
| `first-principles-deconstruction` | Dismantle any problem to bedrock truths and rebuild from scratch. |
| `idea-velocity` | Generate 50+ ideas in a single pass — zero-judgment first phase. |
| `lateral-thinking-engine` | Apply de Bono's lateral thinking — random entry, provocation, challenge. |
| `reverse-brainstorm` | Invert the problem: guarantee failure, then flip each failure mode into a solution. |

## When to use this category

- "Write a prompt for GPT-4 / Claude / Midjourney / ..."
- "Improve this prompt" / "make it more specific"
- "Brainstorm ideas" / "think outside the box"
- "Apply first-principles thinking"
- "Review this prompt for safety"
