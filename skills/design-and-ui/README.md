# design-and-ui

Skills for **frontend design, UI styling, brand identity, design systems, and creative assets**. Whatever the visual surface — landing page, dashboard, banner, logo, theme — start here.

## Skills in this category

### Frontend & component craft

| Skill | What it does |
| --- | --- |
| [`frontend-design`](./frontend-design/SKILL.md) | Build distinctive, production-grade frontend interfaces. Web components, pages, artifacts, posters. Avoids generic AI aesthetics. |
| [`impeccable`](./impeccable/SKILL.md) | Redesign, critique, audit, polish, harden, animate, colorize, or otherwise *improve* an existing frontend. UX review, hierarchy, accessibility, motion, micro-interactions, anti-patterns. |
| [`ui-styling`](./ui-styling/SKILL.md) | shadcn/ui (Radix + Tailwind), Tailwind utility-first styling, dark mode, accessible components (dialogs, dropdowns, tables), canvas-based visuals. |
| [`ui-ux-pro-max`](./ui-ux-pro-max/SKILL.md) | The encyclopedic UI/UX intelligence — 50+ styles, 161 palettes, 57 font pairings, 161 product types, 99 UX guidelines, 25 chart types, across 10 stacks. |

### Design systems & tokens

| Skill | What it does |
| --- | --- |
| [`design-system`](./design-system/SKILL.md) | Three-layer token architecture (primitive → semantic → component), CSS variables, spacing/typography scales, component specs. |
| [`design`](./design/SKILL.md) | The all-in-one design skill — brand identity, design tokens, logo generation (55 styles), Corporate Identity Programs (50 deliverables), HTML decks, banners, icons, social photos. |
| [`brand`](./brand/SKILL.md) | Brand voice, visual identity, messaging frameworks, asset management, brand consistency, style guides. |
| [`theme-factory`](./theme-factory/SKILL.md) | 10 pre-set themes (colors + fonts) you can apply to any artifact — slides, docs, reports, HTML pages — or generate a new theme on the fly. |
