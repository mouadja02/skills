# api-backend

Skills for **API design, backend services, TypeSpec, OpenAPI, FastAPI, and general server-side development**.

## Skills in this category

| Skill | What it does |
| --- | --- |
| `typespec-api-operations` | TypeSpec API operations — CRUD, pagination, filtering, error models. |
| `typespec-create-agent` | Create AI agents using TypeSpec-defined APIs. |
| `typespec-create-api-plugin` | Build API plugins with TypeSpec schema. |
| `openapi-to-application-code` | Generate application code from OpenAPI specs. |
| `aspnet-minimal-api-openapi` | ASP.NET Core Minimal API with OpenAPI — endpoints, validation, docs. |
