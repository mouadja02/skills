# java-kotlin

Skills for **Java, Kotlin, and Spring Boot** development.

## Skills in this category

### Java

| Skill | What it does |
| --- | --- |
| `java-add-graalvm-native-image-support` | Add GraalVM native image compilation to Java projects. |
| `java-docs` | Javadoc generation, documentation comments, API docs. |
| `java-junit` | JUnit 5 testing — test lifecycle, assertions, parameterized tests. |
| `java-refactoring-extract-method` | Extract method refactoring patterns in Java. |
| `java-refactoring-remove-parameter` | Remove parameter refactoring in Java. |
| `java-springboot` | Spring Boot application development — starters, configuration, REST APIs. |
| `javax-to-jakarta-migration` | Migrate from javax.* to jakarta.* namespace (Java EE → Jakarta EE). |

### Kotlin

| Skill | What it does |
| --- | --- |
| `kotlin-springboot` | Kotlin + Spring Boot — idiomatic Kotlin patterns, coroutines, Spring integration. |

### Spring Boot project scaffolding

| Skill | What it does |
| --- | --- |
| `create-spring-boot-java-project` | Scaffold a new Spring Boot project in Java. |
| `create-spring-boot-kotlin-project` | Scaffold a new Spring Boot project in Kotlin. |

### Testing

| Skill | What it does |
| --- | --- |
| `spring-boot-testing` | Spring Boot testing — MockMvc, @SpringBootTest, Testcontainers. |

## When to use this category

- "Write Java unit tests" / "Spring Boot REST API"
- "Migrate javax to jakarta"
- "Add GraalVM native image"
- "Scaffold a Spring Boot project in Kotlin"
