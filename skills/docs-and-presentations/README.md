# docs-and-presentations

Skills for **producing documentation, slide decks, diagrams, and project artifacts** — anything where the deliverable is a *document*, a *deck*, or a *visualization*.

## Skills in this category

### Written docs

| Skill | What it does |
| --- | --- |
| [`readme-writer`](./readme-writer/SKILL.md) | Write, generate, update, or improve a README.md. Onboarding docs for Python tools, FastAPI backends, React apps, CLIs, dbt projects, monorepos, agent systems. |
| [`project-docs`](./project-docs/SKILL.md) | Technical documentation *beyond* a README — architecture docs, technical design documents, ADRs, API references, data pipeline docs, integration specs, AGENTS.md / CLAUDE.md files. |
| [`project-development`](./project-development/SKILL.md) | Scoping & structuring an LLM project end-to-end — pipeline architecture, agent-assisted development, cost estimation, choosing between LLM and traditional approaches. |

### Presentations

| Skill | What it does |
| --- | --- |
| [`slides`](./slides/SKILL.md) | Strategic HTML presentations with Chart.js, design tokens, responsive layouts, copywriting formulas, and contextual slide strategies. |
| [`pptx`](./pptx/SKILL.md) | Anything involving a `.pptx` file — read, parse, extract text, create, edit, modify, combine, split, work with templates / layouts / speaker notes / comments. |
| [`investor-pitch-deck-designer`](./investor-pitch-deck-designer/SKILL.md) | A complete playbook for structuring, designing, and refining pitch decks tailored for Seed and Series A VC rounds. |

### Visual

| Skill | What it does |
| --- | --- |
| [`svg-diagrams`](./svg-diagrams/SKILL.md) | Diagrams, charts, architecture visualizations, flows, pipeline maps, data models, sequence diagrams. Agent workflows, ETL pipelines, dbt lineage, API flows, system integrations. |

## When to use this folder

- "Write / update the README"
- "Document this architecture / pipeline / API"
- "Create a slide deck for X"
- "Read / edit a .pptx file"
- "Draw a diagram of this flow"
