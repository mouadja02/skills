# devops

Skills for **CI/CD, containers, infrastructure-as-code, Linux operations, and DevSecOps**.

## Skills in this category

### CI/CD & pipelines

| Skill | What it does |
| --- | --- |
| `ci-cd-pipeline-builder` | Build CI/CD pipelines — GitHub Actions, GitLab CI, Azure Pipelines. |
| `devops-rollout-plan` | Plan phased rollouts — canary, blue-green, feature flags. |
| `dependabot` | Configure Dependabot for automated dependency updates. |
| `dependency-auditor` | Audit project dependencies for vulnerabilities and license issues. |

### Containers

| Skill | What it does |
| --- | --- |
| `docker-development` | Docker development workflow — Dockerfile, compose, multi-stage builds. |
| `multi-stage-dockerfile` | Write optimized multi-stage Dockerfiles. |
| `containerize-aspnet-framework` | Containerize legacy ASP.NET Framework apps. |
| `containerize-aspnetcore` | Containerize ASP.NET Core apps for production. |

### Infrastructure as code

| Skill | What it does |
| --- | --- |
| `terraform-patterns` | Terraform best practices — modules, state, workspaces. |
| `terraform-azurerm-set-diff-analyzer` | Analyze diffs in Terraform azurerm provider sets. |
| `helm-chart-builder` | Build and validate Helm charts for Kubernetes. |
| `import-infrastructure-as-code` | Import existing infrastructure into Terraform/Bicep. |

### Linux & system ops

| Skill | What it does |
| --- | --- |
| `arch-linux-triage` | Arch Linux system triage and troubleshooting. |
| `centos-linux-triage` | CentOS / RHEL system triage. |
| `debian-linux-triage` | Debian / Ubuntu system triage. |
| `fedora-linux-triage` | Fedora system triage. |
| `lsp-setup` | Set up Language Server Protocol in editors and IDEs. |
| `sandbox-npm-install` | Safely install npm packages in sandboxed environments. |
| `image-manipulation-image-magick` | Batch image processing with ImageMagick. |
| `saas-scaffolder` | Scaffold a SaaS application with standard DevOps setup. |

### Security & incidents

| Skill | What it does |
| --- | --- |
| `secret-scanning` | Scan for leaked secrets — git history, env files, config. |
| `threat-model-analyst` | Threat modeling — STRIDE, attack surface, mitigations. |
| `data-breach-blast-radius` | Assess the blast radius of a data breach or exposure. |
| `incident-commander` | Lead incident response — triage, comms, postmortem. |
| `observability-designer` | Design observability stacks — metrics, logs, traces, SLOs. |

### Senior roles

| Skill | What it does |
| --- | --- |

| `senior-secops` | Senior SecOps engineer — security operations, SOC, threat hunting. |
