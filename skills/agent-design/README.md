# agent-design

Skills for **designing, architecting, and scaffolding AI agent systems** — single-agent and multi-agent, structured autonomy, BMad Method, browser automation, and RAG.

## Skills in this category

| Skill | What it does |
| --- | --- |
| `ai-agent-builder` | General-purpose agent architecture — tool calling, autonomous loops, subagent orchestration, evaluation pipelines. |
| `multi-agent-patterns` | Supervisor, swarm, handoffs, context isolation, parallel execution. |
| `tool-design` | Design and name agent tools, MCP tool patterns, consolidation strategies. |
| `bdi-mental-states` | Belief-Desire-Intention architecture, RDF-to-beliefs transforms, neuro-symbolic integration. |
| `latent-briefing` | Cross-agent memory without summarization, KV cache compaction. |
| `ai-team-orchestration` | Bootstrap a multi-agent dev team, sprint plans, brainstorm with distinct agent voices. |
| `durable-agent-workflows` | Durable, resumable long-running agent workflows. |
| `structured-autonomy-generate` | Generate structured outputs from autonomous agents. |
| `structured-autonomy-implement` | Implement structured autonomy patterns. |
| `structured-autonomy-plan` | Plan structured autonomy workflows. |
| `create-agent-tui` | Build terminal UI agents. |
| `create-headless-agent` | Create headless/background agents. |
| `create-agentsmd` | Generate AGENTS.md files for agent documentation. |
| `agentic-browser-automation` | AI-powered browser agents using LLM reasoning + Playwright. Self-healing selectors. |
| `agentic-rag-architect` | Design advanced RAG systems with agentic capabilities, GraphRAG, query routing. |
| `bmm-skills` | BMad Method — AI-driven agile: analysis, planning, solutioning, implementation phases. |
| `ai-ready` | Make any repo AI-ready — AGENTS.md, copilot-instructions.md, CI templates. |
| `agent-governance` | Governance, safety, and trust controls for agent systems. |
| `agent-owasp-compliance` | Check agents against OWASP Agentic Security Initiative Top 10. |
| `agent-supply-chain` | Verify supply chain integrity for agent plugins and tools. |
| `trading-agent-builder` | Build trading agents with market data integration. |

## When to use this category

- "Design an agent architecture" / "build a multi-agent system"
