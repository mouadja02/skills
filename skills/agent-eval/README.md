# agent-eval

Skills for **evaluating, improving, and adding memory to AI agents** — autoresearch loops, eval-driven development, and agent memory systems.

## Skills in this category

| Skill | What it does |
| --- | --- |
| `auto-memory-pro` | Curate Claude Code's auto-memory into durable project knowledge. |
| `autoresearch` | Autonomous iterative experimentation loop — define goals, loop, measure, keep/discard. |
| `autoresearch-agent` | Autonomous experiment loop that optimizes any file by a measurable metric. |
| `agentic-eval` | Self-critique, reflection loops, evaluator-optimizer pipelines, rubric-based scoring. |
| `eval-driven-dev` | Evaluation-driven development for agent outputs. |
| `memory-merger` | Merge and deduplicate agent memory files. |
| `local-ai-stack` | Run a full local AI stack — Ollama, Open WebUI, local embeddings. |
| `semantic-kernel` | Microsoft Semantic Kernel — plugins, planners, memory, embeddings. |
